# kq-education

kq education

## Pre-requisites

1. Node JS >= 20
2. NPM

## Installation

1. Clone the repository

  - git clone https://github.com/username/kq-education.git

2. Install dependencies

  - npm install

3. Create a .env file

  - Upate database details and SMTP details

4. Run the project

  - npm run dev
